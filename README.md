# EV-Market-Analysis-PowerBI
EV &amp; Hybrid Vehicle Market Analysis Dashboard
Project Overview

This project analyzes Electric Vehicle (EV) and Hybrid market trends in India to support data-driven decision-making and expansion strategy.

Tools Used
Power BI
PostgreSQL
Power Query
DAX
Key Insights
Identified top-performing states based on EV adoption
Analyzed sales trends and market growth
Evaluated revenue and penetration metrics
Provided insights for strategic expansion
Features
Interactive dashboard
KPI tracking (Sales, Revenue, Growth)
Dynamic filtering and drill-down
